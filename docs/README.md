# CodeGrowthKH Docs

This folder contains local project notes and supporting documentation for the CodeGrowthKH platform.

## Files

- `PROJECT_OVERVIEW.md` high-level summary of the frontend and backend
- `API_OVERVIEW.md` important backend endpoints and route groups
- `ENVIRONMENT.md` local setup and environment variable notes
- `DEPLOYMENT.md` deployment notes, including the current Render backend setup

## Note

This `docs/` folder is currently ignored by Git from the root `.gitignore`, so these files stay local unless you remove the ignore rule.
